# EVO App - ZeroMQ Server

This project was created on top of ZeroMQ Swift made by our folks from [Zewo](https://github.com/Zewo).

It is a simple ZeroMQ server and do not require too much configuration.

## Requirements
- Swift 4.0.3
- Xcode 9+
- SwiftEnv
